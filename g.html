Game Mode Wizard (under construction)

Select the options you want, and it will calculate the correct game mode (Use game select while the game is not running to change the mode):

-- 0 -- -- 1 --
Cycle Levels: The game will start at the level defined by the difficulty switches and change every checkpoint. Fixed Level: The game will start at the level defined by the difficulty switches and not change.
Constant Traffic Pattern: Every time the game start, the same pattern of cars is used. This gives you a full consistent experience and allows memorization to some extent. Random Traffic Pattern: Every time the game start, a different pattern of cars is used. This removes the memorization factor and gives a slightly different experience every time.
Default Speed Difference (Lines): The speed difference between traffic lines is balanced. High Speed Difference (Lines): The speed difference between traffic lines is high. This makes the game harder, overtake windows are shorter and changing lines more difficulty.
Predefined Traffic Intensity: Each level has it predefined, taylor made traffic intensity Random Traffic Intensity: The traffic intensity varies randomly every checkpoint. This makes every play session to vary widely and is not suitable from competitive play. It does not affect each level checkpoint time.
