Game Mode Wizard

Select the options you want, and it will calculate the correct game mode (Use game select while the game is not running to change the mode):

-- 0 -- -- 1 --
Cycle Levels: The game will start at the level defined by the difficulty switches and change every checkpoint. Fixed Level: The game will start at the level defined by the difficulty switches and not change.
Constant Traffic Pattern: Every time the game starts, the same pattern of cars is used. This gives you a full consistent experience and allows memorization to some extent. Random Traffic Pattern: Every time the game starts, a different pattern of cars is used. This removes the memorization factor and gives a slightly different experience every time.
Default Speed Difference (Lines): The speed difference between traffic lines is balanced. High Speed Difference (Lines): The speed difference between traffic lines is high. This makes the game harder, overtake windows are shorter and changing lines is more difficult.
Predefined Traffic Intensity: Each level has it predefined, taylor made traffic intensity. Random Traffic Intensity: The traffic intensity varies randomly at every checkpoint. This makes every play session to vary widely and is not suitable for competitive play. It does not affect each level checkpoint time.

Game Mode: 0000 = 0

Controllers

A quick reference the controllers and switches:

Important: Braking and accelerating at the same time brakes with half intensity.

Input Game Paused Game Running

Reset Change cars / Apply difficulty switches Restarts the current game
Joystick Direction Changes the game border Left, Right: Turns the car / Up: accelerates / Down: Break

Game Over Screen

Gives you a detailed statistics of the game session:

  • S: Score / Distance in decimal.
  • T: The total in seconds.
  • G: Time gliding in seconds.
  • H: The number of hits.
  • C: Total number of checkpoints.
  • Unique Identifier: Game Mode (0 to F), Car Type (0 to 3), D. Switch 1 (B - A), D. Switch 2 (B - A), | Game Version. Example: E2AB|1.42
